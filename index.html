This is my cheat sheet for Display in CSS

Property Value Default Description
Display inline, block, flex, inline-block, grid, none inline Specifies the layout behavior of an element
Visibility visible, hidden visible Specifies whether the element is visible or not
opacity [0.0, 1.0] 1.0 Specifies the transparency of an element
box-sizing content-box, border-box content-box Specifies how the size of an element is calculated
overflow visible, hidden, scroll, auto visible Specifies how an element behaves when its content overflows its container
position static, relative, absolute, fixed static Specifies the type of positioning of an element
float left, right, none none Specifies how an element flows within its container
clear both, left, right, none none Specifies whether the element should prevent overlapping elements
object-fit fill, contain, cover, none, scale-down fill Specifies how an image is displayed within an element
object-position [x-offset] [y-offset] 50% 50% Specifies the position of an image within an element
z-index [numeric value] auto Specifies the stacking order of an element in the z-space
display: table table, table-cell, table-row, table-caption table Specifies how an element is displayed as a table or table cell
table-layout auto, fixed auto Specifies how the width of table cells is calculated
